E-commerce

Sistema e-commerce simple con microservicios reactivos

Mayo 2024

Este proyecto fue desarrollado durante el curso de Sring WebFlux Masterclass. Consta de 3 microservicios, Usuario, Producto y Pedido. El usuario podrá consumir los recursos brindados por cada microservicio, ya sea para darse de alta en el sistema, como así también, realizar una compra y ver las compras que ha realizado.

Tecnologías utilizadas: Java, SpringBoot, Spring WebFlux, Docker, PostgreSQL, MongoDB
