Monopatines

Aplicación para Monopatines Eléctricos

Octubre 2023

Esta aplicación fue requisito para aprobar la materia Arquitecturas Web de TUDAI. Basada en microservicios, fue construida para responder a la siguiente problematica: Una empresa quiere lanzar un negocio para permitir el alquiler de monopatines electrónicos en distintas paradas de una ciudad capital, para lo cual requiere el desarrollo de una aplicación móvil para los usuarios del servicio, y adicionalmente una aplicación Web para la gestión correspondiente...

Tecnologías utilizadas: Java, SpringBoot, Spring Security, Docker, MySQL, MongoDB, Swagger
