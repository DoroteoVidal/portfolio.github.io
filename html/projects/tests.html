JUnit y Mockito

Sistema de banco y cuentas

Abril 2024

Este proyecto fue realizado durante el curso de Spring Boot Test con el fin de realizar distintos tipos de test a la aplicación, desde tests unitarios hasta tests de integración.
Tambien se puso en práctica las estrategias de desarrollo, TDD (Test Driven Development) y BDD (Behavior Driven Development).

Tecnologías utilizadas: Java, SpringBoot, JUnit, Mockito
